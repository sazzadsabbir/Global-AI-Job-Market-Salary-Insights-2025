# Excel-Based Analysis & Interactive Dashboard
This project explores global trends in AI job demand, salaries, skills, and industry growth using a structured dataset. It includes both an in‑depth analysis and an interactive Excel dashboard designed to present key insights clearly and intuitively.

## Project Structure
- **Dashboard/** – Interactive Excel dashboard visualizing AI job and salary trends  
- **Analysis/** – Data cleaning, exploration, and insight generation

## Summary
The project highlights how AI roles vary across regions, industries, and experience levels while revealing salary patterns, top skills, and emerging job trends.
